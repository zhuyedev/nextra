# 关于 Nextra

Nextra 最初由 [Vercel](https://vercel.com) 成员 [Shu Ding](https://twitter.com/shuding_) 和 [Paco Coursey](https://twitter.com/pacocoursey) 在2020年创建。自2021年起，[Yixuan Xu](https://twitter.com/yixuanxu94) 对项目做出了巨大贡献。到了2022年，[The Guild](https://the-guild.dev) 的 [Dimitri Postolov](https://twitter.com/dimaMachina_) 加入核心团队，协助开发 2.0 版本。

## Team

目前，该项目由 [Shu Ding](https://twitter.com/shuding_) 和 [Dimitri Postolov](https://twitter.com/dimaMachina_) 维护。您可以在 [GitHub](https://github.com/shuding/nextra/graphs/contributors) 上查看完整的贡献者列表。
