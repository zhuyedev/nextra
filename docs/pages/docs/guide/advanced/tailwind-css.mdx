# Tailwind CSS

Tailwind CSS 是一个 CSS 框架，提供了一组预定义的 CSS 类，以快速样式化元素。您可以按照官方的[Tailwind CSS for Next.js 文档](https://tailwindcss.com/docs/guides/nextjs)为您的 Nextra 项目设置 Tailwind CSS。

要在 Markdown 文件中使用 Tailwind 类，您还需要将 `.md` 和 `.mdx` 文件添加到 `tailwind.config.js` 中的 `content` 列表：

```js filename="tailwind.config.js" /md,mdx/
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './pages/**/*.{js,jsx,ts,tsx,md,mdx}',
    './components/**/*.{js,jsx,ts,tsx,md,mdx}',

    // Or if using `src` directory:
    './src/**/*.{js,jsx,ts,tsx,md,mdx}'
  ],
  theme: {
    extend: {}
  },
  plugins: []
}
```
