# 自定义 CSS 支持

Nextra 完全兼容 Next.js 的[内置 CSS 支持](https://nextjs.org/docs/basic-features/built-in-css-support)，包括 `.css`、`.module.css` 和 Sass (`.scss`, `.sass`, `.module.scss`, `.module.sass`) 文件。

例如，考虑以下名为 `styles.css` 的样式表：

```css filename="styles.css"
body {
  font-family: 'SF Pro Text', 'SF Pro Icons', 'Helvetica Neue', 'Helvetica',
    'Arial', sans-serif;
  padding: 20px 20px 60px;
  max-width: 680px;
  margin: 0 auto;
}
```

您可以创建一个 `pages/_app.js` 文件，并在其中导入您的 CSS 文件：

```jsx filename="pages/_app.js"
import '../styles.css'

// 在新的 `pages/_app.js` 文件中需要这个默认导出。
export default function MyApp({ Component, pageProps }) {
  return <Component {...pageProps} />
}
```

要了解有关 Next.js 中 CSS 支持的更多信息，请查看 [Next.js 文档](https://nextjs.org/docs/basic-features/built-in-css-support).
