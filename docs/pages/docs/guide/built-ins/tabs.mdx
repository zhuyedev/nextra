# 标签切换组件

## 示例

import { Tabs } from 'nextra/components'

{/* prettier-ignore */}
<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>**pnpm**: 快速且节省磁盘空间的包管理器。</Tabs.Tab>
  <Tabs.Tab>**npm** 是一个用于 JavaScript 编程语言的包管理器。</Tabs.Tab>
  <Tabs.Tab>**Yarn** 是一个软件包系统。</Tabs.Tab>
</Tabs>

## 用法

### Default

{/* prettier-ignore */}
```mdx
import { Tabs } from 'nextra/components'

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>**pnpm**: 快速且节省磁盘空间的包管理器。</Tabs.Tab>
  <Tabs.Tab>**npm** 是一个用于 JavaScript 编程语言的包管理器。</Tabs.Tab>
  <Tabs.Tab>**Yarn** 是一个软件包系统。</Tabs.Tab>
</Tabs>
```

### 默认选中标签

您可以使用 `defaultIndex` 属性设置默认选中的标签。

```mdx /defaultIndex="1"/
import { Tabs } from 'nextra/components'

<Tabs items={['pnpm', 'npm', 'yarn']} defaultIndex="1">
  ...
</Tabs>
```

这样 `npm` 就会作为默认选中的标签。

{/* prettier-ignore */}
<Tabs items={['pnpm', 'npm', 'yarn']} defaultIndex="1">
  <Tabs.Tab>**pnpm**: 快速且节省磁盘空间的包管理器。</Tabs.Tab>
  <Tabs.Tab>**npm** 是一个用于 JavaScript 编程语言的包管理器。</Tabs.Tab>
  <Tabs.Tab>**Yarn** 是一个软件包系统。</Tabs.Tab>
</Tabs>
