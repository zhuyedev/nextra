# 步骤条组件

一个内置组件，用于将编号列表转换为按步骤展示。

## Example

<Steps>

### 这是第一步

这是第一步的描述。

### 这是第二步

这是第二步的描述。

### 这是第三步

这是第三步的描述。

</Steps>

## 用法

将一组 Markdown 的 h3 标题用 `Steps` 组件包裹，以将它们转换为视觉步骤。

<Steps>

### 步骤 1

步骤 1 的内容。

### 步骤 2

步骤 2 的内容。

</Steps>
