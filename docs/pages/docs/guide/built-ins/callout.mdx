# 提示框组件

一个内置组件，用于向读者展示重要信息。

import { Callout } from 'nextra/components'

## Example

<Callout>
  **提示框** 是一段用于吸引注意力的简短文字。
</Callout>

<Callout type="info">
  **提示框** 是一段用于吸引注意力的简短文字。
</Callout>

<Callout type="warning">
  **提示框** 是一段用于吸引注意力的简短文字。
</Callout>

<Callout type="error">
  **提示框** 是一段用于吸引注意力的简短文字。
</Callout>

## API

`Callout` 组件接受以下属性：

### `type` （可选）

提示框的类型。

- 类型：`'default' | 'info' | 'warning' | 'error'`
- 默认值：`'default'`

### `emoji` (optional)

在提示框中显示的图标。您也可以使用此属性显示自定义图标。

- 类型：`string | ReactNode`
- 默认值：`'💡'`

### `children`

提示框的内容。

- 类型：`ReactNode`
