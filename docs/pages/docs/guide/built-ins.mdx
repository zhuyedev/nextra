# 内置组件

Nextra 包含一些内置组件，您可以使用它们来更好地美化您的内容：

<Cards>
  <Card
    icon={<WarningIcon />}
    title="提示框"
    href="/docs/guide/built-ins/callout"
  />
  <Card icon={<CardsIcon />} title="标签切换" href="/docs/guide/built-ins/tabs" />
  <Card
    icon={<IdCardIcon />}
    title="卡片"
    href="/docs/guide/built-ins/cards"
  />
  <Card icon={<OneIcon />} title="步骤条" href="/docs/guide/built-ins/steps" />
  <Card
    icon={<FolderTreeIcon className="h-5 w-5" />}
    title="文件树"
    href="/docs/guide/built-ins/filetree"
  />
</Cards>
