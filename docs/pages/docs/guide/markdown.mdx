# Markdown

## MDX

在 Nextra 中，位于 pages 目录下的所有 `.mdx` 文件将通过 [MDX](https://mdxjs.com/about) 渲染，它是一种支持 React 组件的高级 Markdown 格式。

例如，您可以在 Markdown 文件中这样导入并使用 React 组件：

```mdx filename="Markdown"
## Hello MDX

import { useState } from 'react'

export function Counter({ children }) {
  const [count, setCount] = useState(0)
  return (
    <button onClick={() => setCount(count + 1)}>
      {children}
      {count}
    </button>
  )
}

<Counter>**Clicks**: </Counter>
```

最终会生成：

import { Shadow } from '@components/utils/shadow'
import { useState } from 'react'

export function Counter({ children }) {
  const [count, setCount] = useState(0)
  return (
    <button onClick={() => setCount(count + 1)}>
      {children}
      {count}
    </button>
  )
}

<div className="mt-6 rounded-xl border border-gray-200 p-4 dark:border-gray-900">
  <Shadow mode="open">
    <h2>Hello MDX</h2>
    <Counter>**Clicks**: </Counter>
  </Shadow>
</div>

除了基本的 MDX，Nextra 还内置了一些高级 Markdown 功能。

## GitHub 风格的 Markdown

[GFM](https://github.github.com/gfm) 是由 GitHub 创建的 Markdown 扩展，支持删除线、任务列表、表格等功能。

### 自定义标题 ID

您可以使用格式 `## 我的标题 [#自定义-id]` 指定自定义标题 ID。例如：



```md filename="Markdown"
## Long heading about Nextra [#about-nextra]
```

在这个例子中，`#about-nextra` 将作为标题链接使用，替换默认的 `#long-heading-about-nextra`。
