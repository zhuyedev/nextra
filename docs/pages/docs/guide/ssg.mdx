# Next.js SSG

使用 Next.js，您可以通过[静态生成 (SSG)](https://nextjs.org/docs/basic-features/pages#static-generation-recommended)预渲染页面。您的页面将在构建时生成，并静态提供给访客。它还可以通过 CDN 缓存，以最大限度地提高性能。

Nextra 也支持这一点。以下是一个示例：

上面的数字是在构建时通过 getStaticProps 生成的。启用[增量静态重渲染](https://nextjs.org/docs/basic-features/data-fetching#incremental-static-regeneration)后，它将保持最新。

---

以下是上述示例的 MDX 代码：

```mdx
import { useData } from 'nextra/data'

export const getStaticProps = ({ params }) => {
  return fetch(`https://api.github.com/repos/shuding/nextra`)
    .then(res => res.json())
    .then(repo => ({
      props: {
        // 我们在页面 props 中添加一个 `ssg` 字段，
        // 该字段将提供给 Nextra 的 `useData` 钩子。
        ssg: {
          stars: repo.stargazers_count
        }
      },
      // 页面将每 60 秒被视为过期并重新生成。
      revalidate: 60
    }))
}

export const Stars = () => {
  // 从 SSG 获取数据，并将其渲染为组件。
  const { stars } = useData()
  return <strong>{stars}</strong>
}

Nextra has <Stars /> stars on GitHub!
```
