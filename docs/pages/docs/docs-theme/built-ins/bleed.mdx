import { Bleed } from 'nextra-theme-docs'

# Bleed 组件

## 示例

当使用 `<Bleed>` 包裹您的内容时，它将比容器稍微宽一些，并在两侧溢出。

<Bleed>
  <div className="bg-white dark:bg-neutral-800 px-16 py-10 text-center border">
    _写作没有什么特别的。你只需要坐在打字机前，**然后写就是了**。_

    — 欧内斯特·海明威

  </div>
</Bleed>

当您希望展示一些图形信息时，它提供了更好的阅读体验，这些信息通常在较大尺寸下看起来更好。

例如，您可以将文本、图像、视频或任何组件放入其中：

<Bleed>
  <iframe
    className="aspect-video w-full"
    src="https://www.youtube.com/embed/3hccXiXI0u8"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowFullScreen
  />
</Bleed>

您甚至可以使用 `<Bleed full>` 使其完全溢出：

<Bleed full>
  <>![风景](https://source.unsplash.com/eaxwP9J_V6s/1600x398)</>
</Bleed>

## 用法

```mdx
import { Bleed } from 'nextra-theme-docs'

<Bleed>嘿，我可以在这里使用 **Markdown** 语法。</Bleed>

<Bleed full>
  ![风景](https://source.unsplash.com/eaxwP9J_V6s/1600x398)
</Bleed>

<Bleed full>
  <iframe
    src="https://codesandbox.io/embed/swr-states-4une7"
    width="100%"
    height="500px"
    title="SWR-States"
  />
</Bleed>
```
